Process of using:

(1) Since we don't have the database of thapar students right now
(2) git clone <repo>
(3) pip install django
(4) be in the folder where the file manage.py exist
(5) Now run python manage.py createsuperuser
(6) It will ask for a username (just enter if you want 'root' as your username)
(7) Then password which when you enter it will be looking like that nothing is being entered but in reality you are entering a password. Just enter the password and in confirm password do the same thing
(8) You can leave the other details like email.
(9) Your superuser account is ready to use.
(10) Now run python manage.py runserver it will give you a local host link http://127.0.0.1:8000/
(11) Open it on browser as http://127.0.0.1:8000/admin
(12) This will open up the admin page where you enter the same username and password you created as superuser.
(13) Inside under Site Administration you can see a option of Students under Library click it.
(14) This will open up a page where you could see the name of all our team members, there will be button named ADD Student click it and it will open a page where you have to enter name, email(thapar one) and roll no.
(15) Click Save and now you can close the admin page.
(16) http://127.0.0.1:8000/ open this and this will take you to the main page, there login with the email(thapar one) and rol no. you entered in Admin section.
(17) It will take you to our main page and rest process of our site will be explained in the video.
